# 失眠怎么办, 硬躺还是起床

[refer](https://mp.weixin.qq.com/s/i6jJSvaqdqXmYXbfCO5lTw)

**结论: 果断起床!**

- 硬躺并不会让你得到优质的休息，如果你着急睡着而变得焦虑的话反而会增加你的脑力消耗，让你第二天更加困倦。

- 起来后，两件事别做：刷手机和看时间

- 那起来做什么呢？

  可以选的活动有很多，比如:
  
  - 阅读纸质书籍
  
    建议看非虚构类的书籍，不建议看以前从未看过的小说，因为情节太刺激。推荐看理财、职场成长、心理学等书籍。如果你是孕妈，建议看科学育儿书籍，可以为宝宝做知识储备。
  
  - 正念冥想
  
  - 听轻音乐或白噪音
  
  - 去阳台看夜空

**总结**: 

- 床最好只用来睡觉。
- 睡不着就起床，困了再睡。
- 晚上不要看时间。 
- 早上固定时间起。



# 马太效应 Matthew Effect

- **强者愈强, 弱者愈弱**, 源自圣经新约的马太福音。在马太福音第25章当中有这样一句话：

  “凡有的，还要加给他叫他多余。没有的，连他所有的也要夺过来。”

- 对立: 回归平均

  核心: 一个偶然的小概率事件，很难重复出现.

- 那么，什么情况下，马太效应的规律会占主导，什么情况下，回归平均的规律会占主导呢？让我们举一个例子：

  

  有一个明星，他在一个月之内迅速爆红，这是马太效应在短期内的影响。10年之后，已经没多少人记得他了，这是回归平均在长期的影响。

  如果这个明星的走红主要是依靠运气，那么他很快就会回归平均；如果他主要是依靠着真本事，他的回归平均就会迟一些，更多的享受到马太效应的好处。

  这个世界，还算是大体公平的，有这两大规律的共同作用，才使得强者不可能永远强大，才使得我们普通人拥有了逆袭的机会。

# 应届生职场入门毒打指南

[refer](https://mp.weixin.qq.com/s/Oouvn3dyAMcy-KoPm8rElA)

- 只要不违法，大大方方为自己谋求利益，是最合理不过的事情了。

  学生思维的一大局限就是不肯从功利的角度思考问题，觉得谈功利是可耻的，这就搞错了，功利从来都不可耻，可耻的是超越法律底线的功利.

  工作这件事情，你要把所有人都当做是竞争对手。

  要知道这个世界的资源是有限的，职场很多时候就是我多吃一口你少吃一口的事情，放弃无畏幻想，为了自己的利益，撸起袖子努力奋斗。

  

- 所谓职业规划，就是让你为了长期的钱，舍弃短期的钱。

  对应届生而言，在面临工作选择决策的时候，有6个需要关注的点，需要综合评估。依照优先级从高较低排列分别是行业，城市，公司地位，岗位性质，直属领导的SB程度和薪酬

  

- 行业

  应届生选行业，要看横向和纵向。
  

  

  横向分为朝阳行业，夕阳行业和坑比行业.

  朝阳行业代表这个行业正在蓬勃发展，人才缺口巨大，存在溢价，或者说存在一定泡沫。

  夕阳行业代表的是这个行业的格局和利益已经被分配完成，人才已经不再稀缺，能找到工作或许不难，但是想要高工资就别想了。
  
  坑比行业代表的是这个行业完全不缺人，行业也不怎么赚钱，甚至还在不断淘汰已有的人，你再怎么优秀也很难找到对口的专业
  
  
  
  纵向是竞争充分行业和竞争不充分行业。
  
  充分竞争是指这个行业的薪酬是存在较大变数的，同业公司对人才的渴求较高，并且猎头较为活跃，让你跳槽的时候有的跳。
  
  竞争不充分是指这个行业基本上大家都差不多，也没啥好跳的，一进来也没太有的选，一个萝卜一个坑。
  
  
  
  如果是纯粹的靠自己奋斗，那尽可能的往**朝阳行业和竞争充分的行业**去凑，只有在这种行业里，个人才更能把握自己的命运，才有可能获取更大的收益。
  
  如果是家里有相关的资源，应该是往具有资源的竞争不充分的行业去凑，因为在这种行业里，你的资源可以更容易的变现，存量市场更加会为资源开出更高的价格。
  
- 城市

  尽可能选择**高速发展，人口高速流入的地区**.

  因为只有在这些地区中，冒出来的各种机会才会越来越多，相关基础设施也会更加成熟，市场化程度和开放化程度也会更高。

  

  另外就是在大城市里，没有人会在意你是谁，在干什么，这对于年轻人而言是一种相对的自由。

  哪怕你在发达地区未必保证能留下来，因为房子还是比较贵的，那也可以考虑在发达地区赚钱，然后攒钱在周边或者老家省会留下来，这个其实不冲突。

  

  可能还有很多毕业的朋友面临和另一半异地的抉择问题，可能另一半会要求你跟着他/她放弃更好的机会去小地方过。

  如果让我选感情和事业，选100次也是100次事业，因为事业是自己的，感情却是不确定的，人更是会变的。

  要把主动性掌握在自己手里。

- 公司地位和岗位职能

  哪怕小厂开出更高的工资，都建议不要太犹豫，**尽可能选大厂**。

  第一份工作，公司是越大越好，行业地位越高越好，因为对于初出茅庐的你而言，你除了一纸文凭什么价值都没有，你最重要的是让自己变得更加值钱，以及稳定落地。

  一个行业最顶级的人才大部分都聚集在大公司，在大公司里你可以接触到行业最先进的内容，最完善的流程，相对最友好的新人体验，很多大公司都会专门有老带新的机制，老员工会认领新人，可以帮你更加稳妥的从学生形态转化为社会人形态。

  

  为什么要把岗位拿出来和公司规模放在一起呢？

  第一, 大厂虽然好, 同时要结合岗位来看

  跳槽时候的价格，是公司背书+项目履历+职位类型的，公司背书很重要，但不是唯一点。

  选择岗位的原则是，依照核心岗位以及岗位门槛来选，**越核心，门槛越高的岗位，越有选择的价值**。

  技术类的岗位最优先，然后是产品类，然后是运营类的，然后是行政类和销售类。

  第二，很多人对于大厂和小厂的认知存在问题。

  很多人对于大厂的认知可能单纯是牌子大或者规模大，但这不完全是。

  要看规模，更要**结合行业影响力**来看。

  有时候一些规模不那么大，但是行业地位领先的公司，其效能是不低于大公司的。

  而有些大公司的边缘类岗位，也没有想象中那么好。

- 直属领导的SB程度

  衡量老板是否可以追随，主要看两个点:

  第一个是**许下的承诺能否兑现**。

  第二个是有没有**经常用梦想和荣誉给你洗脑**。

- 薪酬

  虽然对应届生，薪酬的数字本身不是特别大的重点，大家主要看的是三年后。

  但关于薪酬，还是有个点要注意。

  那就是**领导或者HR和你谈论薪酬时的态度**。

  判定的标准就是在他们问你还有什么要问的时候，可以问问他们的规划以及相关的应届生培养机制，最好能给到具体的案例。

  如果你能够得到非常明确且细致的回答，那么OK。

  如果你得到的是非常笼统且敷衍的回答，那么你就要谨慎了。

# 命如逆水行舟，不进则退

> [refer](https://mp.weixin.qq.com/s/QaUoAC0wzpQ42Qj48HTL3Q)

- "35岁危机"

  有些职业的价值在短时间内飞快地爆发出来，但随着时间的流逝，开始逐渐衰减。

  对企业来说，一个大前期职业，利益最大化的选择就是快速榨干，然后抛弃。

  很多互联网从业者并没有清晰地意识到自己是占了职业的便宜，还以为这辈子都能拥有远高于其他行业的收入水平，于是养成了把后半辈子的钱也一起花掉的消费习惯。

  他以为自己的人生是一柱擎天，但其实只是抛物线。

  本质上，这才是陷入中年危机的根源。

- 前期职业的一个典型特点是，市场需求的极度不稳定性，供需容易出现大量变动。

  互联网是一个典型的前期行业，在行业发展的早期，有大量的市场可以开拓，涌现了大量互联网独角兽企业，各种概念各种赋能各种烧钱，做的是增量市场。
  
  一方面供不应求，一方面市场涌入了大量的热钱，企业也给的起钱，这就造就了互联网行业远高于其他行业的起薪。
  
  但是在单一领域活下来的企业就那么几家，而倒闭的那些公司离开的程序员只能再次投入就业市场。
  
  与此同时，市场上还有大量计算机专业的毕业生在涌来。
  
  等到新市场被分割得差不多了，开始做存量的时候，大家忽然发现，市场其实并不需要这么多程序员。
  
- 前期职业还有一个特点，就是可替代性强，属于劳动密集型工种。

  稍微复杂的需求和架构优化以及算法肯定做不了，但大多数公司也不需要程序员做这些。如果没有了个人经验和技术的优势，那么身体素质随年龄的衰弱，就会直接降低自身对企业的价值。

- 这是一个吃得快跑得快的游戏

  - 在这个红利期多赚钱多存钱，降低物欲，增加抗风险能力.

    然后也不要急着让家人全职在家，不管怎样也要有个双保险在。

  - 多吃资源，增加个人价值，提高企业的替代成本。

    有时候社会竞争的本质，就是比拼谁能在短时间吃下更多的资源，然后快速消化，再去抢更多的资源。


# 提升工作效率

[摘自](https://mp.weixin.qq.com/s/zH9kFjJQ5zE9mKGEiwEYAA)

- 个人

    1. 持之以恒的学习

        学习方法: 比如要聚焦学和工作上有紧密联系的, 可以更多机会实践; 系统性,构建知识体系; 独立思考, 带入自己的场景深入剖析

    2. 目标导向

        通过目标倒推每阶段要完成的任务, 而不是随波逐流

    3. 情绪管理

        职业化的心态; 尝试转移和宣泄

    4. 学会合理的拒绝

        不是推诿

        不能做老好人: 有底线, 想清楚"我"的价值

    5. **沟通**

        - 总体

            - 克制*随意*沟通

                每个人时间都是宝贵的

            - 高效的提问

                - 不怕问出低级问题, 但要清晰描述

                    背景 -> 问题现状 -> 疑问 -> 需要的帮助

                - 礼仪

                    不要吝啬赞美和感谢

            - 高效的回答

                - 不怕麻烦让提问者重复问题

                    明确自己已经清晰理解了问题

                - 真不懂不要装大神

                    每个人都有知识盲区

            - 条理, 重点突出, 简练

        - 跨部门

            - 谨言慎行

                只确认自己能够确认的

                确保沟通的信息准确

            - 定时反馈

                周期性, **及时**反馈沟通的结论

            - 向上

                风险预警和预期管理 

                及时向上级寻求帮助

                不断兑现"承诺"去建立信任机制

    6. 定期回顾

        - 形成适合自己的工作模式
        - 反思, 总结 -> 进步

    7. 自动化

        重复性工作 机械会比人更靠谱

    8. 优先级管理

        做最重要的事

    9. 善用工具

        - 开发工具/编译工具/调试工具/交流工具/协同工具等
        - 顺手的最好, 但不要故步自封

- 团队

    1. 面对面讨论

        效率最高的沟通方式，对方的表情，语气语态，肢体语言可以提供非常多的信息，帮助我们调整沟通技巧，更好地达到一致的沟通目的.

        - 专注, 控时

    2. 会议

        - 明确主题

        - 主动参与

            不做无关的事, 效率也不高

        - 纪要

            输出+回顾

            大部分的工作会议是要对齐信息差以及达成某种共识，最终是要收敛到结论输出上。对于结论，最好是以文字化的记录，以邮件或者讨论组中输出。在某些时候，口头共识 ≈ 口说无凭，切记。

        - 质量管理

            - 短期的阵痛会带来长期的收益
            - 帮助他人也是成就自己

        - 与团队保持一致

            木桶理论. 补齐短板，让自己至少不是团队的短板，追求成为长板

- 外部

    1. 邮件

        - 分类/打标签
        - 分批分时段处理
        - 发送邮件前最好读上几遍确认【可延迟发送】：错别字有没有，数据图表是否准确，附件是否完整，干系人是否覆盖等等

    2. 即时消息

        - 定期查聊天框

        - - 是否遗漏消息
            - 有些对话你看你当时看了，没有想到怎么回复，所以没有回，如果没有标记或者回顾，到后来你可能你都记不起来了。

        - 如何回复消息

        - - 图+文字【条理性：1，2，3】+链接+@人
            - 这大概是 IM 交流中最值得注意的一点了。为什么需要图+文字+链接？图片显然可以传递更直观的信息，文字可以描述现象或者问题，链接可以给对方便捷跳转指示。
            - 需要意识到，工作沟通不同于个人社交沟通，工作沟通为业务服务，我们关注的大家高效地解决问题而不是社交寒暄，所以要尝试学会用简洁的话语去表达核心诉求。交流中，发大段描述性文字，只有截图没有文字描述等情形要避免，这对提高双方的效率都大有裨益。

    3. 电话

        一种同步的沟通方式，它的互动性强，响应快，但是对工作的打断较多，对个人的工作效率影响大

        - 目的明确

        - - 想好要沟通的事项，比如可以先列张表

        - 过程中，确保双方的理解是一致的

        - - 必要时让对方重复

        - 控时

        - - 达到目标就收手，减少时间占用

        - 结论

        - - 文字化/邮件化
            - 类似于上面的”会议“那里提到，沟通结论一定要文字化，这既是职业的表现，也是契约的保证。电话沟通又没有录音，说好的事儿有遗漏怎么办？没说好的事儿后来对方说有又怎么办？
